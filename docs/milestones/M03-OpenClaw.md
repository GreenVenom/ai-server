## Objective

Deploy OpenClaw as the primary AI orchestration platform for local and cloud models.

### Deliverables

* Docker deployment
* Local Ollama integration
* Claude provider configuration
* Multi-model routing
* Conversation persistence
* Authentication
* Configuration backup
* Health checks
* Operational documentation

### Success Criteria

* OpenClaw starts automatically.
* Local models are accessible.
* Claude is available as a fallback provider.
* Multiple local models can be selected.
* Configuration is backed up.
* Health checks pass.
* Documentation is complete.
