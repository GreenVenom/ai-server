# M04 - Qdrant

## Objective

Establish Qdrant as the local vector database and retrieval foundation for the platform.

## Scope

M04 will define and validate Qdrant deployment, persistent storage, collection lifecycle, and embedding integration.

## Deliverables

- Documented Qdrant deployment and data conventions.
- Validated collection and retrieval lifecycle.

## Validation

Validation criteria will be finalized with the M04 implementation plan.

## Exit criteria

M04 is complete when its documented deployment and retrieval validation pass.
