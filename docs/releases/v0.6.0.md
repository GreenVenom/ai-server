# v0.6.0 Release Notes

## Summary

v0.6.0 completes M06 by adding approved Obsidian retrieval and platform inspection through two local stdio MCP servers. Sandboxed OpenClaw agents receive exactly eight read-only tools.

## Highlights

- Four Obsidian retrieval and four platform-inspection tools with strict schemas and sanitized errors.
- Explicit OpenClaw tool filters and sandbox allowlists enforce least privilege.
- Platform inspection uses predetermined commands with `shell=False`, a restricted environment, timeouts, and bounded output.
- Unit, integration, abuse, policy, and ten-scenario agent acceptance tests cover the service boundary.
- Status, health, and verification checks validate an inventory of two servers, eight tools, and zero diagnostics.

## Known limitations

The MCP services are local stdio integrations only; they do not bind a network port. The Tailscale CLI can report its version as `unknown`; runtime health is checked separately through process inspection.

## Next steps

M07 will add monitoring for runtime visibility, service health, benchmark trends, resource use, and alerting.
